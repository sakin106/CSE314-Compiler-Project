# CSE314 Custom Language Compiler

This repository contains a purely table-driven Lexical Analyzer (DFA) and an LL(1) Predictive Parser built in C from scratch. 

## Features
* **Zero-String Matching Lexer:** Token classification operates entirely through an 85x43 state transition matrix in $O(1)$ time complexity.
* **LL(1) Predictive Parser:** Validates syntax using a 15x19 mapping table across a 24-production Context-Free Grammar.
* **Live Terminal UI:** Interactive console design for real-time code testing.
