# CSE314-Compiler-Project
This C-based compiler frontend for CSE314 features a purely table-driven Lexical Analyzer and an LL(1) Predictive Parser. Built by Sakin Bin Jabed &amp; team, it uses a zero-string matching 85x43 DFA matrix for O(1) tokenization and a 24-production CFG. It includes a live terminal UI for real-time syntax validation.
